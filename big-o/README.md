# Big O

Big O allows us to determine the time complexity of certain algorithms and thus compare them objectively.

| Type        | Function       | Result (for _n_ = 100)                    |
| ----------- | -------------- | ----------------------------------------- |
| Constant    | O(1)           | 1                                         |
| Logarithmic | O(log _n_)     | ≈7                                        |
| Linear      | O(_n_)         | 100                                       |
| Logarithmic | O(_n_ log _n_) | ≈664                                      |
| Quadratic   | O(_n_^2)       | 10,000                                    |
| Exponential | O(2*n*)        | 1,267,650,600,228,229,401,496,703,205,376 |
| Factorial   | O(_n_!)        | ≈9.332622 × 10157                         |
